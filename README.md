## 自動化測試流程

自動編譯 + 自動輸入測資
將輸出與其他檔案存到指定資料夾
比對輸出與答案並輸出 `AC` 或 `WA`
> 若 `WA` 會輸出相異處

1. 下載 `test.sh` 和 `.env.example` 並改成 `.env`
2. 將以下三個路徑貼進 `.env` 中
    - FILES: 待編譯檔案路徑
    - SAMPLE_DIR: 測資資料夾路徑
    - OUTPUT_DIR: 存放生成出的 `.out`, `.exe`, `.class` 的資料夾路徑
3. 終端機執行 `./test.sh`
